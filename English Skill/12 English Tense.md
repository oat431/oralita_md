
## Simplify Table

| Form \ Time        | Past                  | Present                  | Future                        |
| ------------------ | --------------------- | ------------------------ | ----------------------------- |
| Simple             | S + V2                | S + V1(s/es)             | S + will + V1                 |
| Continuous         | S + was/were + Ving   | S + Vbe + Ving           | S + will + be + Ving          |
| Perfect            | S + had + V3          | S + has/have + V3        | S + will + have + V3          |
| Perfect Continuous | S + had + been + Ving | S + has/have + been + V3 | S + will + have + been + Ving |
## Symbol
S: Subject
V1 : Base Form
V2 Past Form
V3: Past Participle
Vbe: Verb to be (is/am/are)

## Explain Time
Past : เกิดไปแล้ว
Present : กำลังจะเกิด
Future : คาดว่าจะเกิด

## Explain Form
Simple : ข้อเท็จจริง หรือ เรื่องทั่ว ๆ ไป
Continuous: เหตุการณ์ที่กำลังจะเกิดขึ้น / หรือ เหตุการณ์เฉพาะช่วงเวลานั้น ๆ 
Perfect: สิ่งที่เสร็จไปแล้ว
Perfect Continuous: การกระทำที่ยาวนาน และมีผลต่อเนื่อง
